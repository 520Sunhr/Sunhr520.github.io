contest tool(以下简称 ct)

预期

可以自定义一个名字，然后自动创建一系列的文件夹， 每个文件夹内部有一个对拍器，然后有答题模板（DIY） 并且可以初步实现启动对拍器和使用特定的输入输出

功能列表

  • ct [name] 创建一个名字为 [name] 的 文件夹
  • ct config 进行设定 设定内容 : 对拍器模板，答题模板，是否开启对拍器，调试时的数据文件的名称
  • ct init 已当前exe所在位置进行初始化: 需要同目录下有 dpq.cpp tem.cpp data.cpp
  • ct test 进行调试
  • ct dp 使用对拍器进行对拍
  • ct help 使用帮助
  • ct version 版本查看

注意

  1. config与init都是作用与同个文件，故选其一即可
  2. dp与test应当在题目目录(a\b\c…)下使用
  3. 本exe并非直接运行 请配置环境变量(path中添加一个exe所在文件夹)后用命令行使用
  4. 生成的 ctConfig 为配置文件如若删除后请使用 config或者init 重新配置
  5. 作者第一次写,很多功能都不完善(鸡肋),怼的时候麻烦轻点
