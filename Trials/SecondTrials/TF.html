描述

BaoBao 打 CF 的时候看到一个函数:f(x,y)=(x|y)-y,其中|表示按位或。
BaoBao想要进一步研究一下这个函数的性质，于是有了下面的问题。
BaoBao有一个长度为n的数组，a1、a2、...an，
他想要找到一个整数x是的f(a1,x),f(a2,x)...f(an,x)的值都为0.
但是显然有很多个x符合要求，
因此BaoBao想要你帮他找到一个最小的满足要求的x。
快来帮BaoBao解决掉这个问题吧，相信这个简单的问题一定难不倒你~。

输入

第一行一个整数T表示有T组测试数据，对于每组测试数据：
第一行一个整数ｎ表示数组长度为ｎ
第二行ｎ个数，分别表示a1,a2......an
1 < T < 10
1 < n < 2e5
0 < ai < 2^60

输出

输出一个整数，表示答案

提示

当x分别取0~13是数组的情况
0: 4 5 8
1: 4 4 8
2: 4 5 8
3: 4 4 8
4: 0 1 8
5: 0 0 8
6: 0 1 8
7: 0 0 8
8: 4 5 0
9: 4 4 0
10: 4 5 0
11: 4 4 0
12: 0 1 0
13: 0 0 0

题目分析

有 n 个数 ，这 n 个数与一个 x 按位或 后 等于 x
换句话说， n 个数 有 1 的位置 x 也为 1 
例如：
    4： 0 0 1 0
    7： 1 1 1 0
    9： 1 0 0 1 
所以说按照上面的结论 
    x： 1 1 1 1 == 15

inline long long int read()
{
    long long int ans = 0, mark = 1;
    long long int t = getchar();
    while (t < '0' || t > '9')
    {
        if (t == '-')
            mark = -1;
        t = getchar();
    }
    while (t <= '9' && t >= '0')
    {
        ans = (ans << 3) + (ans << 1) + t - '0';
        t = getchar();
    }

    return ans * mark;
}


typedef long long int ll;
const int N = 2e5 + 5;
